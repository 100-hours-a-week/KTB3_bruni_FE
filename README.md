## 🌈프로젝트 소개
Bruni's Board 는 오픈 커뮤니티 웹 애플리케이션입니다. 누구나 글과 사진을 통해 공유하고 이야기를 나눌 수 있는 서비스입니다.   

<br>   

## 📌주요 기능   

| 기능 | 설명 |
| --- | --- | 
| **사용자** | •회원가입/로그인/로그아웃<br>•회원정보 수정/탈퇴<br>•인증(쿠키-세션) | 
| **게시판** | •게시글 CRUD (이미지 업로드)<br>•조회수/좋아요/댓글수 표시| 
| **게시글** | •좋아요, 댓글(작성/수정/삭제) |
